B2B register

This module add extra fields to auth sign up page

Setting

Go to Settings > General Settings > Users.

In Customer Account - select option Free .

It allows your new users to get a sign-up option.

Fields

  • Name
  • Company
  • Vat
  • fiscal code
  • Email
  • Phone
  • city
  • Zip
  • Address
  • Country
  • Industry
  • PEC
  • SDI
Note: This module is tested and working in Odoo Community Edition but not tested in Enterprise Edition.
